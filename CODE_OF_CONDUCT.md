# Code of Conduct

This project follows the
[CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md),
which is based on the [Contributor Covenant](https://www.contributor-covenant.org/).

In short: be respectful, be constructive, assume good intent, and keep the project a
welcoming place for contributors of all backgrounds and experience levels.

## Reporting

Violations can be reported privately to the maintainer via the contact options on the
[GitHub profile](https://github.com/stefanlievers) or through a
[private security advisory](https://github.com/stefanlievers/cilium-egress-operator/security/advisories/new)
if no other channel is appropriate. Reports are handled confidentially.
